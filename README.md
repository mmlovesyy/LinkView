# LinkView
仿朋友圈分享链接的自定义 View

<img src="linkview.png" width="600px" />

## 使用方法

```
    <com.mmlovesyy.linkview.LinkView
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:background="@drawable/bg_link"
        android:clickable="true"
        android:paddingBottom="5dp"
        android:paddingLeft="5dp"
        android:paddingRight="5dp"
        android:paddingTop="5dp"
        app:drawableHeight="45dp"
        app:drawableWidth="45dp"
        app:maxLines="2"
        app:srcDrawable="@mipmap/eye"
        app:text="是否经常感到眼睛干涩吗？该如何缓解及防治呢？"
        app:textLeftMargin="6dp"
        app:textSize="12sp" />
```
